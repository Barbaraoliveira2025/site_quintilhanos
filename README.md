Site institucional premium para empresa especializada em drywall, pintura e acabamentos sofisticados em São Paulo.

## Tecnologias
- **HTML5** Semântico
- **CSS3** Moderno e Responsivo
- **JavaScript** Leve e Performático
- **SEO Local** Avançado com Schema.org
- **GitHub Pages** Ready

## Estrutura

```
/
├── index.html           # Página principal com todas as seções
├── style.css            # Sistema de design premium
├── script.js            # Interatividade e otimizações

```
